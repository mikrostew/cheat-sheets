# IntelliJ (& Android Studio)

[//]: # (key symbols from http://osxdaily.com/2012/03/27/making-sense-of-mac-keyboard-symbols/)

### Lines

|||
|---|---|
| duplicate line | `⌘D` |
| delete line | `⌘⌫` (or `⌘Y`) |
| join lines | `^⇧J` |
| move line up | `⌥⇧↑` |
| move line down | `⌥⇧↓` |
| new line below | `⇧⏎` |

### Code

|||
|---|---|
| comment out line/selection | `⌘/` |
| optimize imports | `⌥⌘O` |
| override method | `⌘O` |
| reformat code | `⌥⌘L` |

### Refactoring

|||
|---|---|
| rename | `⇧F6` |
| extract variable | `⌘⌥V` |

### Search / Navigation

|||
|---|---|
| find usages | `⌥F7` |
| next error | `F2` |
